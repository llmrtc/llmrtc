---
title: Core SDK Overview
---

`@metered/llmrtc-core` exposes the types, orchestrator, protocol constants, and helper utilities used by both the backend and web client.

Includes
- Type definitions for providers, messages, tools, playbooks
- `ConversationOrchestrator`
- Protocol version and message constructors

Use it when building custom providers, bespoke backends, or testing orchestration logic in isolation.
