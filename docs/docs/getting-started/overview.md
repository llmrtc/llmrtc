title: Getting Started Overview
---

Use this section to stand up a working voice assistant quickly.

**What you’ll do**
- Install the backend and web client packages
- Run a minimal OpenAI + ElevenLabs stack
- Understand the end-to-end event flow
- Try local-only mode with Ollama + FasterWhisper + Piper
- Optionally run the Vite demo (`examples/vite-demo`) or the 80-line minimal example (`examples/minimal`)

**Prerequisites**
- Node.js 20+
- API keys (OpenAI + ElevenLabs) for cloud path, or Ollama/FasterWhisper/Piper for local
- FFmpeg installed if you enable streaming TTS

Next: [Installation](installation) → [Backend quickstart](backend-quickstart) → [Web client quickstart](web-client-quickstart).
